# ESP32-S3 五路灰度循迹小车

## 简介
本项目基于 **ESP32-S3-WROOM** 主控，使用 ESP-IDF + FreeRTOS 开发，配合 LM339 电压比较器、五路灰度传感器、L298P 电机驱动模块，实现经典黑线自动循迹功能。

代码采用**分层架构 + 状态机(FSM)** 设计，驱动与业务逻辑解耦，结构清晰，适合嵌入式入门学习与单片机课程实训。项目预留 HC-SR04 超声波传感器接口，可二次开发实现避障功能。

## 硬件组成
| 器件 | 型号 | 功能 |
| ---- | ---- | ---- |
| 主控 | ESP32-S3-WROOM | 主控制器，逻辑运算与任务调度 |
| 循迹模块 | 五路灰度传感器 + LM339 | 赛道识别，黑线输出高电平(1)，白纸输出低电平(0) |
| 电机驱动 | L298P | 直流电机驱动，控制电机正转、反转、停止 |
| 动力单元 | 直流减速电机 × 2 | 小车行走机构 |
| 扩展模块 | HC-SR04 超声波传感器（待开发） | 障碍物检测、避障功能扩展 |
| 电源 | 锂电池组 | 整车供电 |

## 代码架构
整体分为三层，低耦合易维护：
1. **底层驱动层**
  - `gray_sensor`：五路灰度传感器采集驱动
  - `motor_l298p`：L298P 电机 GPIO 初始化与方向控制
2. **逻辑决策层**
  - `track_fsm`：循迹状态机，根据传感器数据输出行驶指令
3. **应用主程序**
  - `app_main`：程序入口，循环执行状态检测与电机控制

## 功能列表
### 已实现
- 五路灰度传感器数据实时采集与串口打印
- 六状态循迹逻辑：直行、小幅左右修正、急转、停止
- L298P 电机方向控制（正转 / 反转 / 停机）
- 可配置主循环刷新周期，默认 80ms

### 待扩展
- HC-SR04 超声波避障驱动开发
- LEDC PWM 电机调速、弯道减速优化
- 循迹 + 避障组合行驶逻辑

## 引脚说明
所有引脚通过头文件宏定义统一管理，改接线只需修改宏，无需改动业务逻辑：
- 灰度传感器：`GRAY_PIN_1 ~ GRAY_PIN_5`
- 右电机控制：`MOTOR_A_IN1`、`MOTOR_A_IN2`
- 左电机控制：`MOTOR_B_IN1`、`MOTOR_B_IN2`
- 电机使能：`L298P_ENABLE_A`、`L298P_ENABLE_B`（默认拉高使能）

## 开发环境
- 框架：ESP-IDF 5.0+
- 内核：FreeRTOS
- 编译工具：VS Code + ESP-IDF 插件 / 命令行
- 编程语言：C
